Introdução ao MVC

O padrão MVC (Model-View-Controller) é uma arquitetura de software utilizada no desenvolvimento utilizada no desenvolvimento de aplicações
para separar a lógica de negociós, a interface do usuário e o controle dos dados.

Componentes do MVC

  • Model (modelo) - representa os dados e a lógica de nogócio da aplicação
  • View (Visão) - Responsável pela interface como usuário e exibição dos dados
  • Controller (Controlador) - intermedia a comunicação entre o Model e View
